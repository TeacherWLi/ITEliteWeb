Hyper Text Markup Language
超文本标记语言(Hyper Text Markup Language)，标准通用标记语言下的一个应用。 是网页制作所必备的一种标记语言 (markup language)。
Cascading Style Sheets
层叠样式表(Cascading Style Sheets, CSS)是一种用来表现HTML或 XML等文件样式的计算机语言。
JavaScript
JavaScript是一种动态类型、弱类型、基于原型的语言。用来给HTML网页增加动态功能， 广泛应用于各类客户端。
JQuery
jQuery是一个快速、简洁的JavaScript框架，是继Prototype之后又一个优秀的JavaScript代码库（或JavaScript框架）。
